## 사다리 요구사항 및 구현목록
### 기능 요구사항
* 사다리 실행 결과를 출력해야 한다.
* 개인별 이름을 입력하면 개인별 결과를 출력하고, "all"을 입력하면 전체 참여자의 실행 결과를 출력한다.

### 구현 목록
* [ ] 실행 결과 입력 기능
* [ ] 실행 결과와 사용자 매핑 기능
* 실행 결과 출력 기능
    * [ ] 결과 보고 싶은 사람 출력 기능
    * [ ] 모든 사람 결과 출력 기능 

### 예외처리 목록
* [ ] 참가자 수와 실행결과 개수와 맞지 않을 경우 예외 처리
* [ ] 결과 보고 싶은 사람이 참가자가 아닌 경우 예외 처리
